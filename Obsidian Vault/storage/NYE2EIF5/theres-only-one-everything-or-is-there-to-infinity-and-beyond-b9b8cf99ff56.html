There’s Only One Everything … Or Is There? To Infinity — and Beyond!

by
Erin Dees

Erin explores the counterintuitive properties of the infinite.

Did y’all catch Russ Olsen’s column entitled “Infinitely Floating Points” in the August 2018 edition of PragPub? In it, he describes how computer representations of floating-point numbers are hopelessly limited, no matter how we model them.

One of these many limitations, as he aptly puts it, is that:

“[T]here are way, way more mathematical real numbers than mathematical integers … “

He goes on to say what this means for computers. But let’s pause for a moment to consider this amazing fact of mathematics he has just unearthed in passing: there’s more than one kind of infinity!

Lots of sets of numbers that one might guess were different sizes are actually the same size. It sounds like there should be twice as many integers as there are even integers, for instance — but these sets are exactly the same size! (Russ shows you why, and I’ll recap the argument in a minute.)

What about real numbers, though, including all the numbers that can’t neatly be written as integers or fractions? They’re a different animal. There are so many more of them that we need to revisit what we thought we knew about infinity.

In this article, we’re going to talk about how there’s more than one infinity. We’re going to do so by playing an imaginary game of wits with legendary mathematician Georg Cantor, who developed these eye-opening concepts. You may think it unfair to pick a fight with a dead man, but — spoiler alert— he’ll win.

How Many of These Are There?

Before we begin, let’s talk for a minute about how we compare sets. We’re going to briefly rewalk the path that Russ laid out for us. We’ll use fruit examples instead of farm animals, though, so this will be totally different.

Intuitively, we know that the set {1, 2, 3} is the same size as the set {apples, pears, bananas}. We can verify this by seeing that there is a one-to-one correspondence between them. (Mathematicians might call this kind of pairing up a bijection, but you and I are engineers, so let’s stick to 1–1 correspondence.) That is, we can arbitrarily pair up one item from each set, with no items left unpaired.

Here’s one way you can do so for these two sets:
(1, apples) (2, pears) (3, bananas)

This isn’t the only way to pear pairs, mind you, just one way. (“Pair pears?” I’m beginning to see why Russ didn’t use a fruit analogy.)

If there are any items left over when you’ve used up one set, then the two sets are different sizes. For instance, the set {1, 2, 3} is smaller than {apples, pears, bananas, kumquats}. No matter how you slice them, you’ll always have a fruit left over; for example:

(1, apples) 
(2, pears) 
(3, bananas)
(???, kumquats)

Make sense? Let’s turn our attention to infinite sets and begin the first round of our game.

Infinity Plus One

In the previous section, we added a fruit to a three-element set and made it bigger. Flush with this realization, we proudly declare that adding an item to any set always makes it larger.

“Not so fast,” cautions Cantor. “Infinite sets can defy these kinds of intuitive guesses.”

“Consider,” he continues, “the set of positive integers {1, 2, 3, …} vs. the set of natural numbers {0, 1, 2, 3, …}. It looks like the set of natural numbers is one item larger. But you can put these two sets into a 1–1 correspondence without leaving any items out. Here’s one example of how to pair them up:

(0, 1) 
(1, 2) 
(2, 3) 
…

“Every natural number on the left has a corresponding positive integer on the right.”

Cantor has just shown us that adding one element to an infinite set doesn’t change its size. Informally, we can use the symbol ∞ to represent the size of an infinite set. We’ll write this finding as ∞ + 1 = ∞.

By the same reasoning, you can add any finite number of items to an infinite set and still be able to make this kind of 1–1 correspondence. So, even if we add a million items, ∞ + 1,000,000 = ∞.

That gives us an idea for our next challenge for Cantor.

Infinity Plus Infinity

“We’re going to do more than add just a few items to our set this time,” we say. “We’re going to add an infinite number of items. Surely, that will result in a larger set.”

“You’d think so,” Cantor replies. “But both sets would still be the same size. Let’s combine, for instance, two subsets of natural numbers, the evens {0, 2, 4, …} and the odds {1, 3, 5, …}. Each of these is an infinite set. But when you combine them, the new set containing all the naturals isn’t any bigger. You can still pair the naturals with, say, the evens:

(0, 0) 
(1, 2) 
(2, 4) 
…

“These three sets — evens, odds, naturals — are the same size. And yet you can build the set of naturals by combining the other two infinite sets.

“In other words, ∞ + ∞ = ∞.”

Looks like Cantor has us beat when it comes to addition. To try to outsmart him, we turn to multiplication next.

Infinity Times Infinity

You can’t exactly multiply two sets like you can multiply numbers. But you can take their cartesian product, where you make a new set out of all possible pairs of items, one taken from each set.

For instance, the cartesian product of {apples, pears} and {1, 2, 3} is:

(apples, 1) 
(apples, 2) 
(apples, 3) 
(pears, 1) 
(pears, 2) 
(pears, 3)

That’s every possible pair, six pairs altogether. The size of the cartesian product (6) is the same as the other two sizes multiplied together (2 × 3).

What happens if we take the cartesian product of the positive integers with themselves?

(1, 1) 
(1, 2) 
(1, 3) 
… 
(2, 1) 
(2, 2) 
(2, 3) 
…

“This must be a larger set than just the positive integers,” we say with a smirk. “We can’t even make an infinite list of all of them; see the gaps in the middle?”

“I can get rid of the gaps,” says Cantor, “so that you can list them alongside the positive integers in a 1–1 correspondence.

“Order the pairs by the total of each pair. So, (1, 1) goes first because its total is 2. Next come (1, 2) and (2, 1), both of which add to 3. Next are (1, 3), (2, 2), and (3, 1), which all add to 4. We’ll sort each subgroup by the first number in it.”

We look at the new 1–1 correspondence made possible by this ordering:

(1, (1, 1)) 
(2, (1, 2)) 
(3, (2, 1)) 
(4, (1, 3)) 
(5, (2, 2)) 
(6, (3, 1)) 
…

No matter what pair of positive integers we arbitrarily choose, we can always find a corresponding single positive integer. The cartesian product of two infinite sets is no larger than either set. Cantor has shown us that ∞ × ∞ = ∞.

By the way, we’ve just accidentally shown that the set of positive integers and the set of positive rational numbers (that is, positive fractions) are exactly the same size! After all, a fraction is just a pair of numbers with a bar between them, such as 2/3.

So, we can rewrite the above list with fractions, like so:

(1, 1/1) 
(2, 1/2) 
(3, 2/1) 
(4, 1/3) 
(5, 2/2) 
(6, 3/1) 
…

Sure, we’ve got some improper fractions in there such as 2/2, but that only proves our point further. There aren’t any more fractions than there are integers. (I’ve only shown you the positive ones, but the same logic applies to the set of all fractions.)

Let’s Get Reals

Thinking one step ahead of Cantor, we say, “We can see where this is headed. We’re going to keep finding sets of numbers, and you’re going to tell us there are just as many natural numbers as there are any other kind of number.”

“That’s not true,” retorts Cantor. “The set of real numbers is bigger. In fact, there are infinitely more real numbers just between 0.0 and 1.0 than there are positive integers.”

“We’ll prove you wrong,” we unwisely say. “We’ll make a list of real numbers between 0.0 and 1.0, and match them up with integers.”

Cantor folds his arms and waits, a hint of a smile visible.

We put on our thinking caps and come up with the following pairing. Most real numbers have decimal representations that go on forever, hence the ellipses.

(1, 0.14159265358…) 
(2, 0.33333333333…) 
(3, 0.71828182846…) 
…

Cantor doesn’t even ask how we came up with the list. “I can think of a number that’s not on your list,” he says. “First, I’ll chop off the leading 0. from each number:

14159265358… 
33333333333… 
71828182846… 
…

“Then, I’ll take just the digits along the diagonal from upper left to lower right:

1…………. 
.3………… 
..8……….. 
…

“I’ll add 1 to each of those digits, wrapping from 9 to 0 if I need to:

2…………. 
.4………… 
..9……….. 
…

“Finally, I’ll write this new sequence of digits as a new decimal number:

0.249…

“This number isn’t anywhere on your list. It differs from the number in the first row, because the first digit is different. It differs from the number in the second row, because the second digit is different. And so on. My number is different from every number in your list.”

Cantor did this without using any knowledge of how we constructed our list. “No matter how you try to build a 1–1 correspondence, I’ll always be able to find a real number that doesn’t have a row in your list and therefore doesn’t have a corresponding positive integer.”

We sigh, defeated but curiously elated. We haven’t won the game, but we’ve won knowledge. Cantor has shown us that one infinitely large set (the reals between 0.0 and 1.0) is larger than another. Cantor’s clever proof technique here is called diagonalization, so named because we chose digits along a diagonal.

More Than One Everything?

The word “infinity” just isn’t precise enough for these situations.

Mathematicians have discovered an entire hierarchy of infinities, known as transfinite numbers.

So far, we’ve been looking at cardinal numbers, that is, numbers that represent sizes of sets. We haven’t even peeped at ordinal numbers, which indicate positions within a sequence (first, second, third, and so on). If you thought cardinal numbers had surprising properties, wait ’til you see transfinite ordinals!

That will have to wait for another day, though. (Would you like to see such an article on another day?) For now, just know that when you enjoy the energetic They Might Be Giants song, “There’s Only One Everything,” you’ll have a retort ready.
